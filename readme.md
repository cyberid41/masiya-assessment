# Quick-Start Guide


## Installation

**1. Clone this repo:**

```sh
git clone https://github.com/cyberid41/masiya-assessment.git
```

**2. Install the dependencies:**

```sh
cd masiya-assessment

composer install
```


**3. Start the development server :**

```sh
php artisan serve
```

> Please open the following url `localhost:8000`
